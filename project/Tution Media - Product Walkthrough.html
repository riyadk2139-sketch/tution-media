Screen 07 · Location · privacy

A guardian's home is never exposed to strangers.

Before approval, only a fuzzed area pin. On approval, exact address and Google Maps turn-by-turn — for that one tutor, that one job. Scoped and revocable.

Posted
Public view is area + landmark + ~300m approximate pin. Enough to estimate commute.
Requested
Tutor taps "Request location." Guardian gets a single tap to approve — one tutor or all shortlisted at once.
Unlocked
Exact pin + Google Maps turn-by-turn unlock in-app. Auto-revoked if tutor isn't hired.
3-in-1
Safety for guardians · anti-leakage (navigation lives in-app) · trust signal on first contact.

Screen 09 · Masked inbox

Numbers stay private.

Every chat and call routes through the platform. Guardians never see the tutor's phone number — and vice versa. Anti-harassment and anti-leakage in one mechanism.

Number masking, both sides
VOIP for calls, in-app for chat. The platform is the comms layer — period.
System messages mark transitions
"Trial booked," "Location approved," "First payment received" — green chips inline in the thread.
WhatsApp / SMS fallback
Critical alerts still send via SMS — device and data realities in Bangladesh are non-negotiable.
